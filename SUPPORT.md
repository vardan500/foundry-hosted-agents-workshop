# Support

## How to file issues and get help

This project uses [GitHub Issues](https://github.com/Azure/foundry-hosted-agents-workshop/issues) to track bugs and feature requests for the samples in this repository. Please search existing issues before filing new ones to avoid duplicates. For new issues, file your bug or feature request as a new Issue.

For help and questions, please refer to:

- [Microsoft Foundry documentation](https://learn.microsoft.com/azure/ai-foundry/)
- [Microsoft Q&A — Azure AI Foundry](https://learn.microsoft.com/answers/tags/387/azure-ai-foundry) for technical questions

## Microsoft Support Policy

Support for this **sample workshop repository** is limited to the resources listed above. The workshop are provided as-is for learning and reference purposes; they are not covered by formal Microsoft Customer Service & Support (CSS) agreements.
