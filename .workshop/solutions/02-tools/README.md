# Solution 02 — TravelBuddy with tools

Adds mock get_weather, get_local_time, and convert_currency functions.

## Run it

1. Copy `.env.example` to `.env` and fill in values.
2. Sign in with Entra ID: `az login`.
3. Install dependencies: `pip install -r requirements.txt`.
4. Start the hosted Responses server: `python main.py`.

Try: `Weather in Lisbon, local time there, and 200 EUR in JPY?`
